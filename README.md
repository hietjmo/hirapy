# hirapy
Romaji to hiragana and katakana

```
# python hirapy.py example-1.txt 
すみません.
あなた は アメリカじん です か?
はい, わたし は アメリカじん です.
にほんご が わかります か?
すこし わかります.
エイゴ が わかります か?
いいえ, エイゴ が わかりません.

Sumimasen.
Anata wa amerikajin desu ka?
Hai, watashi wa amerikajin desu.
Nihongo ga wakarimasu ka?
Sukoshi wakarimasu.
Eigo ga wakarimasu ka?
Iie, eigo ga wakarimasen.

すみません	excuse me
にほんご	japanese
エイゴ	english
わかります	understand
わかりません	don't understand
```
